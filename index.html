FAQ
Common questions
Do I need n8n to use these?

Yes. AutoPacks workflows are built for n8n (self-hosted or n8n Cloud). You'll need a running n8n instance to import and use them. If you don't have one yet, our setup guide walks you through getting started with n8n Cloud in under 10 minutes.

What if I need help setting them up?

Every pack includes a detailed setup guide and video walkthrough. If you still get stuck, you can book a custom implementation session starting at $500 where we'll set everything up for you and walk you through how it all works.

Can I modify the workflows?

Absolutely. You get the raw n8n JSON files. Customize them however you want — add nodes, change logic, integrate with your existing tools. They're yours to modify.

What APIs/services do the workflows connect to?

It depends on the pack. Common integrations include OpenAI/Claude for AI, Twilio for SMS, Google Business Profile, various CRMs (HubSpot, GoHighLevel), Stripe, Shopify, and more. Each pack's page lists the exact services needed.

Do I get updates when workflows are improved?

Yes. When we update a workflow (to accommodate API changes, add features, or improve reliability), you'll get an email with the updated JSON file and a changelog. Lifetime updates are included with every purchase.

Can I resell these to my clients?

Not with a standard purchase. If you're an automation consultant or agency, check out our Consultant License ($99/mo) which includes white-label rights to resell under your own brand.

What's your refund policy?

If you can't get the workflows running within 14 days and our support team can't help resolve the issue, we'll refund your purchase in full. We'd rather you succeed than keep your money.
